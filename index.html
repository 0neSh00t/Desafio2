Tamron SP 15-30mm f/2.8 Di VC USD G2 Lente para Canon
Cubriendo una gama versátil de distancias focales ultra gran angulares, el soporte EF Canon 15-30mm f/2.8 Di VC USD G2 de Tamron mejora este amplio zoom con sofisticadas ópticas, enfoque y estabilización de imagen. Ayudando a realizar imágenes nítidas y claras, el diseño óptico incorpora elementos de cristal asféricos y de baja dispersión para reducir las aberraciones esféricas y cromáticas en todo el rango del zoom. Un trío de recubrimientos, AX, eBAND y BBAR, también se han aplicado para suprimir el reflejo de la lente y el efecto fantasma para mejorar el contraste y la reproducción del color cuando se trabaja en condiciones de luz brillante y retroiluminada.

Sigma 50mm f/1.4 DG HSM Art Lente para Sony A
Rápido y versátil, el Sony A-mount 50 mm f / 1.4 DG HSM de Sigma es una imprimación de longitud normal diseñada para lograr un alto rendimiento óptico mediante la incorporación de tres elementos de vidrio especial de baja dispersión y un elemento asférico de vidrio moldeado. Estos elementos, junto con un diseño óptico avanzado, funcionan para minimizar el destello de coma sagital y las aberraciones cromáticas para proporcionar una alta nitidez y claridad de imagen. El gran diámetro y el sistema flotante de la lente también ayudan a corregir el viñeteado para garantizar un brillo periférico mantenido. También se ha aplicado un recubrimiento súper múltiple a los elementos de la lente para reducir los destellos y las imágenes fantasma para un mayor contraste y fidelidad de color. Además, lo que lo convierte en un objetivo ideal para retratos, la apertura máxima brillante f / 1.4 se empareja con un diafragma redondeado de nueve hojas para beneficiar el enfoque selectivo y las imágenes de poca profundidad de campo con un suave bokeh.
